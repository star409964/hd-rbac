## CycleWorkBase
循环处理
>[info] 启动 静态命令 start
~~~
        $data=["name"=>"mikkle"];
        return CycleHandle::start( $data );
~~~
或者
~~~
        return CycleHandle::start(  );
~~~
>[info] 查看状态 静态命令 status
~~~
        return CycleHandle::status(  );
~~~
>[info] 停止 静态命令stop
~~~
        return CycleHandle::stop(  );
~~~
